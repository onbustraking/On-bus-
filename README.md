# On-bus-
OB बस ट्रैकर एक उन्नत वेब ऐप है जो भारत की स्थानीय बसों की लाइव लोकेशन, रूट और समय-सारणी दिखाता है। Firebase पर आधारित यह ऐप यात्रियों को बसों की वास्तविक समय स्थिति, किराया, सीटें और ट्रिप विवरण देता है। पूरी तरह मोबाइल-फ्रेंडली और हिंदी में उपलब्ध, इसे होम स्क्रीन पर ऐड किया जा सकता है।
